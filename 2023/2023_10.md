## 10/25(wed)

#### CSS

* 要素がはみ出した時にスクロールするようにしたい

```css
overflow: scroll
```

縦もしくは横のみスクロールしたい場合は `overflow-x` / `overflow-y` を使うことで実現できる

```css
// 縦方向のスクロールのみ
overflow: scroll
overflow-x: hidden

// 横方向のスクロールのみ
overflow: scroll
overflow-y: hidden
```

* 縦横比を維持したい

```css
// width 1 / height 1 の比率
aspect-ratio: 1 / 1

// '/ height' が省略された場合は規定値は1になる
aspect-ratio: 1
```
